Privacy Policy for ArmoryBot
Effective Date: February 6, 2026

Summary

ArmoryBot does not collect, sell, or track personal data.
The only information it may store is when a user voluntarily chooses to save favorites.

What the bot does
ArmoryBot accepts a prompt you provide and uploads a picture to Discord so it can be shared in your server.
The image is processed only to complete the upload.
Users may optionally save favorite character settings for convenience.
Data we do NOT collect or store
No personal information beyond what is needed for optional favorites
No message content
No images or files permanently
No IP addresses
No analytics or tracking data
Optional saved favorites

If a user chooses to save favorites, ArmoryBot may store:

The user’s Discord ID
The user’s saved character preferences

This information is stored only to provide the favorites feature and is not used for tracking, advertising, or profiling.

Data sharing

ArmoryBot does not sell or share user data with third parties.
Stored favorites data is only used for the operation of the bot’s features.

Data retention
Images are used only to complete the upload to Discord and are not stored permanently by us.
Saved favorites remain stored until the user requests deletion or the data is otherwise removed by the bot operator.
User rights

Users may request deletion of their saved favorites data at any time.
Upon request, the stored Discord ID and character preference data associated with favorites will be deleted.
